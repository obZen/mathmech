Шифр Цезаря (Алгоритм шифорвания)
Шифр простой цикличиской подстановки на некоторую величину сдвига
y - новый символ, полученный сдвигом
y = (x+k) mod n (n мощность алфавита) => x = (y-k+n) mod n

Взлом шифра
Используются данные частотного анализа
1. Построить LT таблицу частот символов: число повторений символа/одлина строки
   GT Глобальная таблица частот для языка
2. Определить СДВИГ* (k) = 0 < СДИВГ < N arg min \sum_{i=0}^N (GT [i + СДВИГ] - LT[i])^2
